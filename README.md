# 🛡️ IndigoShield - Emergency Safety App

IndigoShield is an Android-based emergency safety app that aims to offer immediate assistance during emergency situations.

Features

-  One-touch SOS emergency button
-  Voice trigger activation ("help" / "indigo")
-  Automatic location sharing via Google Maps link
-  SMS alert sent to guardian
- Automatic emergency call after SOS trigger

Tech Stack

- Kotlin
- Jetpack Compose
- Android SDK
- Firebase Cloud Messaging (FCM)
- SMS Manager API
- Location Manager API

How It Works

1. User presses SOS button OR says trigger word.
2. App fetches current GPS location.
3. SMS with live location link is sent to guardian.
4. Emergency call is triggered.
5. Push notification can also be sent via Firebase.

Team

Developed as part of a hackathon project.

Purpose

IndigoShield is intended to improve personal safety by providing instant emergency communication through smart activation features.
